#git笔记

基本命令：
- git init 初始化目录
-  git add ｛文件名称｝ 放入git仓库
- git commit  -m {提交说明} 把文件提交到仓库
- git status 查看提交状态
- git diff 对比改动
- git log 查看提交日志 （--pretty=｛oneline｝ 只看概要）
- git reset --hard ｛版本号｝ 回滚到具体的版本号
- git reset --hard HEAD^ 回滚（^表示上一次 ^^前两次 HEAD~100 前100次）
